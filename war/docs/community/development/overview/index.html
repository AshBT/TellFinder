Overview

TellFinder is an Eclipse project with server-side Java code and client-side HTML and JavaScript. TellFinder uses Apache Ant for build automation.

Contributing to TellFinder

TellFinder is an active open source product developed and managed by Uncharted Software Inc. We welcome feedback, suggestions and technical contributions.

For access to TellFinder source code on GitHub, contact Uncharted. To contribute to the project, please submit a pull request against the master branch of the source code.

Getting Started

The TellFinder project offers multiple paths to getting started. Contact Uncharted to retrieve the supporting TellFinder data (ad databases and map set), then choose either of the following methods depending on your level of technical experience and available third-party tools:

  • Manual Installation: Manually build and deploy TellFinder using third-party build automation, database management and web server tools.
  • Docker: Build a Docker container that automatically assembles TellFinder in a virtualized environment.
